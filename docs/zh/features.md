## 核心安全功能

### XSS 检测

针对 Vue 应用程序的全面跨站脚本漏洞检测。

#### 功能

- **v-html 检测**：检测 `v-html` 指令的不安全使用
- **模板注入**：识别潜在的模板注入点
- **内联事件处理器**：查找不安全的内联事件处理器
- **路由参数 XSS**：检测路由参数中的 XSS
- **基于 DOM 的 XSS**：识别基于 DOM 的 XSS 模式
- **v-text 安全**：检查不安全的 `v-text` 使用
- **v-bind 安全**：验证属性绑定安全性
- **v-for 循环安全**：识别不安全的循环源

#### 检测模式

```vue
<!-- 易受攻击 -->
<div v-html="userInput"></div>
<div v-bind:href="userUrl"></div>
<div v-on:click="userFunction"></div>

<!-- 安全 -->
<div>{{ sanitizedInput }}</div>
<div :href="safeUrl"></div>
<div @click="safeFunction"></div>
```

```bash

# 平衡模式 - 默认，适用于大多数用例
vue-security-scanner . --performance balanced

# 全面模式 - 生产全面扫描
vue-security-scanner . --performance thorough
```
