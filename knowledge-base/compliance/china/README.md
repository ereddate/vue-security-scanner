# 中国合规性标准

## 📚 概述

中国合规性标准是指在中国境内运营的组织需要遵守的一系列法律法规和技术标准。本目录提供了中国合规性相关的文档，包括 GB/T 标准、网络安全法、数据安全法、个人信息保护法和密码法。

## 🚀 如何使用

1. **了解中国合规性要求**：详细阅读中国相关法律法规和标准的核心要求和前端影响
2. **实施措施**：按照实施指南在前端应用中实现合规性措施
3. **验证合规**：使用提供的检查清单和验证方法确保合规
4. **持续监控**：定期更新合规性措施以适应中国法规的变化
