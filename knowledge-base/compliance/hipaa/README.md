# HIPAA 合规性

## 📚 概述

HIPAA（健康保险可携性和责任法案）是美国的一项法规，旨在保护患者的医疗信息。本目录提供了 HIPAA 相关的合规性文档，包括受保护健康信息保护、安全规则和数据泄露通知。

## 📁 目录结构

```
hipaa/
├── phi-protection/    # 受保护健康信息保护
│   ├── phi-definition.md # PHI 定义
│   ├── phi-storage.md # PHI 存储
│   ├── phi-transmission.md # PHI 传输
│   └── phi-disclosure.md # PHI 披露
├── security-rule/     # 安全规则
│   ├── administrative-safeguards.md # 管理保障措施
│   ├── physical-safeguards.md # 物理保障措施
│   ├── technical-safeguards.md # 技术保障措施
│   └── organizational-requirements.md # 组织要求
└── breach-notification/ # 数据泄露通知
    ├── notification-requirements.md # 通知要求
    ├── timeline.md # 通知时间线
    ├── notification-content.md # 通知内容
    └── breach-response.md # 数据泄露响应
```

## 🚀 如何使用

1. **了解 HIPAA 要求**：详细阅读 HIPAA 的核心要求和前端影响
2. **实施措施**：按照实施指南在前端应用中实现合规性措施
3. **验证合规**：使用提供的检查清单和验证方法确保合规
4. **持续监控**：定期更新合规性措施以适应 HIPAA 的变化
