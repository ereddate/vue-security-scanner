# Vite 安全指南

## 📚 概述

Vite 是一个现代化的前端构建工具，提供了快速的开发体验和优化的生产构建。本目录提供了 Vite 相关的安全指南，包括开发服务器安全、构建选项安全和插件安全。

## 📁 目录结构

```
vite/
├── dev-server/         # 开发服务器安全
│   ├── configuration.md # 开发服务器配置
│   ├── network-security.md # 开发服务器网络安全
│   ├── hmr-security.md # 热模块替换安全
│   └── cors-security.md # 开发服务器 CORS 安全
├── build-options/      # 构建选项安全
│   ├── minification.md # 代码压缩安全
│   ├── asset-handling.md # 资源处理安全
│   ├── source-maps.md # 源码映射安全
│   └── target-browsers.md # 目标浏览器安全
└── plugins/            # 插件安全
    ├── plugin-validation.md # 插件验证
    ├── third-party-plugins.md # 第三方插件安全
    ├── custom-plugins.md # 自定义插件安全
    └── plugin-order.md # 插件顺序安全
```

## 🚀 如何使用

1. **选择 Vite 安全领域**：根据您的应用需求选择合适的 Vite 安全领域
2. **参考安全指南**：查阅相应的安全指南文档
3. **实施安全措施**：按照推荐的安全配置和最佳实践实施安全措施
4. **验证实施**：使用安全检查清单确保安全措施的正确实施
5. **定期更新**：定期更新 Vite 和相关依赖以适应新的安全威胁
