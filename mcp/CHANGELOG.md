# Changelog

## 1.7.0 (2026-01-31)

- 更新依赖项 `vue-security-scanner` 到 ^1.6.0
- 添加性能配置选项（`performanceProfile`, `enableParallelScanning`, `enableIncrementalScanning`, `memoryLimit`）
- 添加 Vue 3.6 支持（`enableVue36Features`, `enableVaporModeScanning`）
- 添加 `exclude` 配置选项
- 添加并行规则引擎关闭功能
- 更新版本号到 1.7.0

## 1.3.1 (2026-01-22)

- Initial release of vue-security-mcp
- Vue Security MCP (Multi-Modal Co-Pilot) Tool
- A tool for real-time security scanning of Vue code when using AI
- Advanced semantic analysis and enterprise-grade reporting
- Support for Vue 3.x
- Integration with AI code generation tools
